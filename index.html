Sportiz

Sportiz est une plateforme de visualisation de données dédiée au sport en France. Elle recense le nombre de licenciés pour chaque fédération sportive entre 2016 et 2023. Grâce à des graphiques et des outils interactifs, Sportiz permet d'explorer facilement les tendances et l'évolution de la pratique sportive en France.

Politique de Confidentialité :

Dans le cadre de ce projet, aucune donnée personnelle ou information sur les utilisateurs n’est collectée ni partagée. Notre approche se fonde uniquement sur l’utilisation de données issues de sources officielles, dans le but de concevoir des graphiques et analyses informatifs. Nous nous engageons à utiliser exclusivement des données provenant de sources externes fiables, telles que celles fournies par l'INJEP, pour la création de contenus à visée pédagogique. Aucune information individuelle ou donnée additionnelle n’est donc ni collectée ni exploitée.

Collecte de données par github :

Ce site est hébergé sur GitHub Pages, un service fourni par GitHub. GitHub peut collecter certaines données personnelles des visiteurs, telles que l'adresse IP, le type de navigateur, le système d'exploitation, et les interactions avec le site. Ces données sont utilisées par GitHub pour assurer le fonctionnement du service et améliorer ses performances. Pour plus d'informations sur les pratiques de collecte et d'utilisation des données par GitHub, veuillez consulter leur Déclaration de confidentialité. Déclaration de confidentialité

Sources :

Les données utilisées dans ce projet proviennent exclusivement de sources fiables, notamment le site de l’INJEP. Elles sont employées uniquement dans un but éducatif et informatif, dans le cadre d’un projet étudiant réalisé à l’IUT de l'Université Gustave Eiffel à Champs-sur-Marne. Ces informations sont strictement réservées à un usage pédagogique et ne sont pas destinées à des fins commerciales.

Tous les éléments graphiques de ce projet ont été conçus par Thomas Veber. Les polices d’écritures “Navada” (par SabrCreative) et Helvetica Neue sont des polices libres d’utilisation pour des usages personnels.
